What is a String?

A string is a peice of text which can be stored in a variable. A string is an incredibly maluable datatype which is capable of being mutated into many other datatypes. One essential method of the string is length. str.length calls the ammount of characters present in any one string. Use a combination of for loops and string.length to solve the following problems.
